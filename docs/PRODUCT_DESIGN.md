# Codex Flow 产品设计基线

## 1. 产品定位

Codex Flow 是面向普通用户与专业创作者的桌面 AI 工作台。用户只需要配置 API Key，即可使用对话、模型、Skills、MCP、定时任务、图片和视频能力。

核心价值不是“提供更多配置”，而是把模型、工具和自动化包装成用户能立即理解的任务。

## 2. 设计目标

1. **首次使用低门槛**：第一次打开后，只完成服务商选择和 API Key 输入即可开始。
2. **任务优先**：首页首先询问用户要完成什么，而不是要求理解模型、Agent、MCP 等概念。
3. **过程可见**：明确展示模型、工具调用、任务进度、Token 和费用。
4. **复杂能力渐进出现**：高级配置不阻挡默认流程，在需要时再展开。
5. **本地优先**：密钥、历史记录、任务配置默认保存在本机。
6. **自动化可信**：定时任务必须能查看下次执行时间、最近结果和失败原因。

## 3. 目标用户

### 普通 AI 用户
- 只想输入 API Key 后直接聊天、搜索或生成内容。
- 不理解 Base URL、MCP、Cron 等技术概念。
- 需要清楚知道花费了多少 Token 和费用。

### 内容创作者
- 高频使用图片、视频、文案和资料整理能力。
- 希望把重复操作保存为模板或定时任务。
- 需要文件导入、批量处理和成果导出。

### 开发者与高级用户
- 需要自定义模型服务商、Skills、MCP 和 CLI。
- 关注执行日志、上下文、权限、工作目录和请求详情。
- 需要图形界面与终端共享同一套配置。

## 4. 信息架构

### 一级导航

| 模块 | 用户问题 | 核心内容 |
| --- | --- | --- |
| 智能对话 | 我现在想完成什么？ | 对话、附件、工具、模型、执行结果 |
| 定时任务 | 哪些事情需要自动完成？ | 任务列表、频率、精确时间、执行历史 |
| 用量统计 | 我用了多少、花了多少？ | 请求、Token、费用、模型占比 |
| 模型中心 | 我可以使用哪些模型？ | 服务商、API Key、模型发现、连通状态 |
| Skills & MCP | 怎样扩展能力？ | 推荐市场、安装、权限、自定义配置 |
| 视频工作室 | 怎样快速完成视频？ | 素材、剪辑提示词、模板、导出 |
| CLI 控制台 | 怎样从终端使用？ | 安装、登录、任务、模型和用量命令 |

### 全局入口
- 新建任务：始终返回智能对话并聚焦输入框。
- 搜索：搜索会话、任务、Skill、文件和设置。
- 通知：展示自动任务完成、失败和更新信息。
- 账户菜单：工作区、数据、安全、外观和更新。
- 服务状态：展示本地服务与模型厂商连接状态。

## 5. 核心用户流程

### 流程 A：首次启动
1. 打开应用。
2. 选择服务商，默认推荐 OpenAI。
3. 输入 API Key。
4. 本地验证连接并自动发现模型。
5. 选择默认模型，或直接接受推荐模型。
6. 进入首页，提供四个可直接点击的任务示例。

失败状态必须区分：密钥无效、网络失败、余额不足、接口地址错误、没有可用模型。

### 流程 B：完成一次任务
1. 输入自然语言任务。
2. 系统根据任务推荐模型与工具。
3. 用户发送请求。
4. 实时显示当前步骤、工具和消耗。
5. 返回结构化结果。
6. 用户可以继续追问、导出、保存提示词或创建定时任务。

### 流程 C：创建定时任务
1. 从定时任务页面创建，或将一次对话转为任务。
2. 输入任务名称和提示词。
3. 选择每天、每小时、每周或高级 Cron。
4. 对每天任务支持时、分、秒。
5. 对每小时任务支持“每小时执行 N 次”。
6. 选择模型、工具、结果去向和失败重试。
7. 创建后立即展示下次执行的绝对时间与倒计时。

### 流程 D：安装能力
1. 浏览或搜索 Skill/MCP。
2. 查看能力说明、来源、安装量、权限和兼容性。
3. 点击安装。
4. 首次使用敏感权限时再请求确认。
5. 安装完成后直接提供“立即试用”提示词。

### 流程 E：图片降级生成
1. 解析图片需求。
2. 优先选择已连接的图片模型或 Image Skill。
3. 如果无图片模型，生成固定尺寸 H5。
4. 在本地浏览器渲染 H5。
5. 截图并裁切为目标尺寸。
6. 返回图片，同时保留 H5 源文件供编辑。

## 6. 页面状态要求

每个功能页面都应具备：
- 初始状态
- 空状态
- 加载状态
- 成功状态
- 部分成功状态
- 失败与重试状态
- 离线状态
- 权限不足状态

## 7. MVP 范围

### P0 必须完成
- 本地安全保存 API Key
- OpenAI 兼容接口
- 模型自动发现与手动模型配置
- 基础对话与流式输出
- 单次与日/月/年 Token 统计
- 本地定时任务
- Skills/MCP 安装与管理
- CLI 基础接口

### P1 增强体验
- 视频插件
- 图片 H5 降级方案
- 任务执行历史与重试
- 多工作区和配置导入导出
- 系统托盘、通知与开机启动

### P2 未来能力
- 多设备同步
- 团队工作区
- 插件审核与市场分发
- 企业权限和费用中心
